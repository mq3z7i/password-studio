## Features

- Generate **random passwords**, **memorable phrases**, or **PIN codes**  
- Adjustable **password length** with a smooth slider  
- Options to include **numbers** and **symbols**  
- Real-time **strength estimation**  
- Copy to clipboard with one click  
- Lightweight and fully **client-side**, no data is ever logged  

---

## Demo

Open the `index.html` file in your browser to start generating passwords.

---

## Usage

1. Clone the repository or download the ZIP.
2. Open `index.html` in any modern browser.
3. Select your password type: **Random**, **Memorable**, or **PIN**.
4. Adjust the **length** slider to set the password size.
5. Toggle **Numbers** and **Symbols** if needed.
6. Click **Copy password** to copy it, or **Refresh password** to generate a new one.
